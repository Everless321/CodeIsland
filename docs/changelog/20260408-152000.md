### Bug Fix: Collaborator tile 跳转总是聚焦第一个终端
- **问题**：两个终端在同一画布、相同 CWD 时，点击任一 session 都跳转到第一个 tile
- **原因**：`resolveCollaboratorTile` 的 CWD 匹配策略遍历 tiles 返回第一个匹配项
- **修复**：新增 3 个更精确的匹配策略（按优先级）：
  1. tmux session name → ptySessionId（已有，保留）
  2. tmux socket path 包含 ptySessionId（新增）
  3. PID 链匹配：从 cliPid 沿进程树向上查找 tile 的 shell PID（新增）
  4. CWD 匹配降为最后手段
- PIDResolver.parentPid 改为 internal 访问级别供 TerminalActivator 调用
