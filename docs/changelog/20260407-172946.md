## 变更内容

### CodeIsland

**SessionSnapshot.swift**
- 新增 `tmuxSocketPath` 属性，从 bridge 上报的 `_tmux` 环境变量中提取 tmux server socket 路径
- 在 `extractMetadata`、`SessionStart` 事件处理、`_env` fallback 三处解析该字段

**TerminalActivator.swift**
- 新增 Collaborator 识别：当 `termBundleId == "com.collaborator.desktop"` 时走专属跳转
- `activateCollaborator`: 激活 Collaborator 窗口 + 异步定位画布 tile
- `resolveCollaboratorTile`: 两级匹配策略
  1. tmux pane ID → `tmux -S <socket> list-panes` → session name `collab-<id>` → ptySessionId
  2. 回退：cwd 匹配（读 canvas-state.json + terminal-sessions meta）
- `collabRpc`: 通过 Unix socket 发送 JSON-RPC 请求（canvas.tileFocus）

### Collaborator

**canvas-rpc.js**
- tileList 响应新增 `ptySessionId` 字段，让外部工具可通过 RPC 匹配终端 tile

## 匹配链路
```
codeisland-bridge 采集 TMUX env + TMUX_PANE
  → SessionSnapshot 存储 tmuxSocketPath + tmuxPane
  → tmux -S <socket> list-panes 查找 pane 所属 session
  → session name "collab-<id>" 提取 ptySessionId
  → canvas-state.json 查找对应 tile
  → RPC canvas.tileFocus 平移画布到 tile 位置
```
