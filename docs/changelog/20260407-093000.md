# Charon MCP 服务器集成

**日期**: 2026-04-07
**类型**: 新功能

## 任务描述

将 Charon（SSH 服务器凭据管理工具）的完整功能合并进 CodeIsland，使其成为 CodeIsland 的内置模块。AI 编码工具可以通过 MCP 协议请求服务器连接信息，用户在刘海面板上直接审批，无需再运行独立的 Charon 应用。

## 待办

- [x] Phase 1: ServerStore — SQLite 数据层（复用 ~/.charon/charon.db）
- [x] Phase 2: MCPServer — HTTP/SSE + JSON-RPC 服务器（127.0.0.1:9800）
- [x] Phase 3: PIDResolver — TCP 连接 PID 反查，自动关联到具体对话
- [x] Phase 4: 审批流集成 — AppState/Models/IslandSurface 扩展
- [x] Phase 5: 刘海面板 UI — MCPApprovalBar 审批卡片
- [x] Phase 6: 设置页 — ServersPage 服务器 CRUD 管理
- [x] Phase 7: ConfigInstaller + AppDelegate 集成
- [x] Phase 8: 创建 changelog 文档

### 架构
```
CodeIsland
  ├─ HookServer (Unix socket)        ← 现有
  ├─ MCPServer (HTTP/SSE :9800)      ← 新增
  │    ├─ list_servers               → 返回服务器名称列表
  │    └─ request_server_info        → PID 关联对话 → 刘海审批
  ├─ ServerStore (SQLite)            ← 新增，~/.charon/charon.db
  └─ Settings → ServersPage          ← 新增
```
